# 🗓️ Holiday Meaning Card (기념일 의미 카드)
> 기념일이나 명절의 의미를 알려주는 인터랙티브 카드 웹페이지

**[🔗 Demo 바로가기](https://Duckcchun.github.io/Holiday-Meaning-Card-Page)**

---
<br>

## 🔍 한줄 소개
CSS 3D 애니메이션과 JavaScript DOM 조작을 활용하여, 기념일의 의미를 사용자의 인터랙션에 따라 동적으로 보여주는 반응형 정적 웹페이지입니다.

---
<br>

## 🧰 기술 스택
- **Language:** HTML5, CSS3, JavaScript (ES6+)
- **Core Tech:**
  - CSS 3D Transforms & Transitions
  - JavaScript DOM Manipulation
  - JavaScript Event Handling
  - Responsive Web Design (Media Queries)
- **Deployment:** GitHub Pages

---
<br>

## ✨ 주요 기능
- **3D 카드 플립 애니메이션:** CSS의 `transform: rotateY(180deg)` 속성과 `transition`을 활용하여, 카드에 마우스를 올리거나 버튼 클릭 시 부드럽게 뒤집히는 3D 효과를 구현했습니다.
- **동적 콘텐츠 렌더링:** JavaScript의 `addEventListener`로 버튼 클릭 이벤트를 감지하고, `DOM 조작` (`getElementById`, `.innerHTML`)을 통해 카드 뒷면의 텍스트와 이미지를 동적으로 변경합니다.
- **반응형 웹 디자인:** CSS `Media Query`를 활용하여, 데스크탑, 태블릿, 모바일 등 다양한 디바이스의 화면 크기에 최적화된 레이아웃을 제공합니다.

## 🚀 설치 및 실행 (로컬)

이 프로젝트는 별도의 빌드 과정이나 라이브러리 의존성이 없는 순수 HTML, CSS, JavaScript로 구성된 정적 웹페이지입니다.

```bash
# 1. 저장소 클론
git clone https://github.com/Duckcchun/Holiday-Meaning-Card-Page.git

# 2. 폴더 이동
cd Holiday-Meaning-Card-Page

# 3. 브라우저에서 index.html 파일 열기
# (별도의 npm install 이나 dev 서버 실행이 필요 없습니다.)
````
